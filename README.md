# Healthcare dashboard UI

## Installation

```bash
yarn
```

### To run

```bash
gulp watch
```

### Gulp tasks

#### For favicon generating

```bash
gulp favicon
```

#### For SASS

```bash
gulp favicon
```

#### For Lighthouse

```bash
gulp lighthouse
```
